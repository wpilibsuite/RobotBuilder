Speed Controller Group

What is it?

Controller for combining multiple speed controllers so they act as one.

Properties

SpeedController1
The first speed controller to add to the Speed Controller Group.
SpeedController2
The second speed controller to add to the Speed Controller Group.
SpeedController3 (Optional)
The first speed controller to add to the Speed Controller Group.
SpeedController4 (Optional)
The first speed controller to add to the Speed Controller Group.
