Speed Controller

What is it?

A speed controller that uses PWM signals to control the speed of the attached motor. There are seven main PWM speed controllers available for use in FRC: Victor, Talon, Jaguar, Victor SP, PWM Talon SRX, Spark, and SD540.

Properties

Type
The type of speed controller to use.
Output Port (PWM)
The PWM port on the roboRIO or MXP that the output is plugged into.
Inverted
Whether to invert the direction of the speed controller.
